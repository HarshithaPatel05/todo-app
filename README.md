# 📝 Todo App (React)

A simple and modern Todo application built using React.  
Users can add, delete, and mark tasks as completed with a clean UI.

## 🚀 Live Demo

👉 https://HarshithaPatel05.github.io/todo-app

## ✨ Features

- ➕ Add new tasks  
- ✅ Mark tasks as completed  
- ❌ Delete tasks  
- 📊 Task counter (Total & Completed)  
- 🎨 Modern responsive UI  

---
